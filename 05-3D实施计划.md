## 二、优化计划

### Phase O-1: 性能优化 ⭐

**目标：100+ 智能体流畅运行**

#### 2.1.1 LOD 系统 (Level of Detail)

根据距离切换模型精度：

| 距离 | 精度 | 说明 |
|------|------|------|
| 0-20 | 高精度 | 完整模型+纹理 |
| 20-50 | 中精度 | 简化模型 |
| 50-100 | 低精度 | Box 模型 |
| 100+ | 隐藏 | 移出渲染 |

**实现文件：**
```
client/systems/lod/
├── lod-manager.js      # LOD 管理器
├── lod- Gebäude.js     # 建筑 LOD
├── lod-tree.js        # 树木 LOD
└── lod-agent.js       # 智能体 LOD
```

#### 2.1.2 实例化渲染 (Instancing)

合并相同材质物体，减少 draw call：

| 物体类型 | 数量 | 优化方式 |
|----------|------|----------|
| 树木 | ~50 | InstancedMesh |
| 花草 | ~100 | InstancedMesh |
| 路灯 | ~20 | InstancedMesh |

**实现文件：**
```
client/systems/instancing/
├── instancing-manager.js  # 实例化管理器
└── merger.js             # 静态物体合并
```

#### 2.1.3 空间网格 (Spatial Grid)

优化附近物体查询和碰撞检测：

```
client/utils/
└── spatial-grid.js        # 空间网格工具
```

**API 设计：**
```javascript
spatialGrid.insert(object, x, z);
spatialGrid.query(x, z, radius);  // 返回范围内的物体
spatialGrid.remove(object);
```

#### 2.1.4 性能监控面板

实时显示 FPS、物体数量、draw calls：

```
client/ui/panels/
└── performance-panel.js    # 性能监控面板
```

### Phase O-4: 智能体动画

**目标：智能体行为更自然**

#### 3.4.1 基础动画

| 状态 | 动画 |
|------|------|
| idle | 站立微动 |
| walking | 行走循环 |
| talking | 身体摆动 |
| thinking | 头上问号 |
| resting | 坐/躺 |

#### 3.4.2 动画状态机

```javascript
agent.states = {
  IDLE: { animation: 'idle', duration: null },
  WALKING: { animation: 'walk', duration: null },
  TALKING: { animation: 'sway', duration: 3000 },
  THINKING: { animation: 'idle', duration: 5000, effect: '?' },
  RESTING: { animation: 'sit', duration: null }
};
```

**实现文件：**
```
client/objects/agents/
├── agent-animator.js    # 动画状态机
└── animations/          # 各状态动画
```
